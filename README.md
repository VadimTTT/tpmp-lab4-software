# tpmp-lab4-software — Воздушный извозчик (вариант 3)

## Project Name
**Воздушный извозчик** — система учёта грузовых вертолётов, экипажей и рейсов в высокогорном районе.

## Description
Приложение на C + SQLite позволяет:
- Управлять вертолётами, экипажем, рейсами (CRUD)
- Контролировать лётный ресурс при добавлении рейсов
- Рассчитывать заработок экипажа (5% / 10% от стоимости рейса)
- Формировать отчёты и статистику (макс. рейсы, макс. заработок)
- Разграничивать доступ (командир / член экипажа)
- Сохранять фотографии вертолётов

## Installation
```bash
git clone [https://github.com/VadimTTT/tpmp-lab4-software.git](https://github.com/VadimTTT/tpmp-lab4-software.git)
cd tpmp-lab4-software
make
Usage
Bash
make run
После запуска войдите как:

commander / commander (полный доступ)

ivanov / ivanov (только свои рейсы и свой вертолёт)

Меню позволяет:

Просмотр налёта и ресурса вертолётов

Рейсы вертолёта за период

Статистику по спецрейсам

Статистику по обычным рейсам

Вертолёт с максимальным числом рейсов

Экипаж с максимальным заработком

Свои рейсы (для члена экипажа)

Добавление рейса с проверкой ресурса (командир)

Расчёт зарплат за период (командир)
